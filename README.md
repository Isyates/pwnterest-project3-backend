### API Endpoints
|Routes|Method|EndPoints|Expected Result|
|------|:---:|:---:|:----|
|Index|GET|/posts|Gets all posts |
|Create|POST|/posts|Creates new posts |
|Update|PUT|/posts/:id|Updates existing posts |
|Delete|DELETE|/posts/:id|Removes posts from database|
